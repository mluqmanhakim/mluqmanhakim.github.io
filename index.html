Machine Learning Project

Personality recognition based on visual cues and facial movement (Master thesis)

Built ML models to predict human personality using PyTorch. The dataset is ChaLearn First Impression. Extracted features (action units, head pose, face movements). Trained LSTM, GRU, and Transformer models and achieved 0.89 (1-MAE) as the main metric in the evaluation.

Object detection

Evaluated 3 popular pre-trained models for object detection in PyTorch, including Faster-RCNN, SSD and RetinaNet. The dataset is COCO. Calculated Intersection over Union (IoU) and Mean Average Precision (mAP) as the main metrics. The best result achieved is 50.9 (mAP) using Faster-RCNN.

Sentiment analysis

Built ML models to classify movie review using TensorFlow. The dataset is IMDB movie reviews. Utilized fastText and subword embedding (BPE). Trained LSTM and GRU models. The best accuracy achieved is 0.87 using fastText.

Machine translation: English to Spanish

Built a simple tranlation machine using Keras and TensorFlow. The dataset is Spanish-English from manythings.org. Trained Transformer model. Achieved 0.86 accuracy in the validation.

Classification of people with breast cancer and frequent pattern mining

Trained some models to classify if a person has a breast cancer using Scikit-learn. Models are SVM, Decision Tree, Random Forest, Neural Network models. The dataset is from University Medical Centre, Institute of Oncology, Ljubljana, Yugoslavia. The best accuracy achieved is 0.7 using Random Forest. Discovered frequent attributes using Apriori and FP-Growth and their associations.

Research

Development of mobile self-monitoring tool prototype based on user-centered design

This study aims to develop a mobile application prototype for self-monitoring. User research was carried out by evaluating the existing web-based application via an interview. Findings from the interview were analyzed and used as requirements to develop the mobile application. By taking into account usability components and design principles, an interactive prototype of a mobile self-monitoring tool was designed for both student and instructor roles.
